Functional Programming

Rhyming Dictionary

Written both in Scala and Python as two different applications, this program is performed by use of a separate pronunciation dictionary. This pronunciation dictionary contains both words and the sounds that make up each word. By taking note of which sounds have vowels and the stress given to said vowels (represented as an integer), rhymes for a given word can be found easily and efficiently as long as the word is in the pronunciation dictionary itself.

Web Api

Parking Ticket Data

A deployable web application that shows and marks the most recent parking tickets handed out in the city of Buffalo. By using an interactable map and the Buffalo Parking Ticket API, hovering over any point on the map displays the date, time, and violation description of that specific incident. Data is collected and parsed from the API via the Python web-framework Bottle, where the the information (kept in a JSON string) is then used to display coordinates on a graph through JavaScript using the Plotly library.

About Me
